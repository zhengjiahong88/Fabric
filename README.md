## 請將以下檔案中的對應字眼替換為你的模組名稱
* `src/main/resources/fabric.mod.json`
  * 第 3 行：將 `"mod_id"` 改為你的模組 ID
  * 第 5 行：將 `"Example mod"` 改為你的模組名稱
  * 第 6 行：將 `"This is an example description! Tell everyone what your mod is about!"` 改為你的模組說明
  * 第 12 行:將 `"https://github.com/zhengjiahong88/Fabric"` 改為你的 GitHub 專案連結
* `build.gradle`
  * 第 25 行：將 `"mod_id"` 改為你的模組 ID
* `gradle.properties`
  * 第 21 行：將 `mod_id` 改為你的模組 ID
* 外加四個名為 `mod_id` 改為你的模組 ID
